Rhino-MATLAB Bridge

22/02/2022

The project presented a significant challenge as it required bridging the gap between disparate software systems—specifically, Rhino with Grasshopper and MATLAB—for an academic research initiative led by an esteemed architecture professor at Istanbul Technical University.

The main challenge was to seamlessly integrate these distinct platforms to facilitate the research process.

The Challenge

To address this challenge effectively, I embarked on a comprehensive research journey to familiarize myself with Rhino, Grasshopper, and MATLAB, even though I had no prior experience with these tools. After gaining a foundational understanding of these technologies, I devised an innovative solution using Python as the central bridge.

The Solution

Flask Application: I developed a Flask application to serve as the intermediary between Rhino with Grasshopper and MATLAB. This application was configured to communicate with a specific Grasshopper component designed for network data exchange.

Python Integration: Leveraging Python's versatility, I utilized a Python module to interact programmatically with MATLAB. This allowed for the seamless transmission of input data to MATLAB and the retrieval of relevant results.

Networking: I established a robust networking framework to enable the smooth transfer of data between the different software environments. This network connectivity formed the backbone of the entire solution, ensuring data exchange and synchronization between Rhino with Grasshopper and MATLAB.

By meticulously implementing this comprehensive solution, I successfully bridged the gap between Rhino with Grasshopper and MATLAB, facilitating the academic research project's objectives. This innovative approach streamlined the research process, enabling efficient data exchange and computation between the two distinct platforms.
