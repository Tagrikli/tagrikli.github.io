MTG Arena Log Monitor

01/03/2021

A client needed a desktop application to monitor the log file of Magic: The Gathering Arena and selectively display game actions in real time. The project required parsing undocumented but human-readable logs to extract meaningful game events.

Requirements

The project had specific requirements:

  • Build a desktop app using Python with Kivy or Tkinter
  • Monitor game log file in real time
  • Parse and display relevant game events:
    • Match Start
    • DeckList for the match
    • Cards drawn by player
    • Cards played by player
    • Cards played by opponent
    • Match End and Match Result

Solution

I developed a Python desktop application that continuously monitored the game's log file, parsing entries to identify and display relevant game events in real time. Working with undocumented log formats required careful analysis and pattern recognition to accurately extract the required information.
