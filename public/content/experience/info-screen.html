Raspberry Pi Information Display

05/09/2023

The main goal of this project was to develop a responsive information display system for a Raspberry Pi, which would be connected to a vertical TV. The client's requirements included the integration of various data inputs onto a single page, necessitating the creation of a backend to fetch data from different services via APIs.

Additionally, the project involved the challenge of displaying an RTSP stream on the screen, which required innovative solutions.

Challenges Faced

Data Integration: Integrating data from multiple sources into a single page presented a significant challenge. Each data source had its own API structure and requirements, requiring meticulous backend development.

RTSP Stream: One of the client's key requirements was displaying an RTSP stream on the Raspberry Pi. However, web browsers do not natively support RTSP, prompting the need for a creative solution.

Solutions Implemented

Technology Stack: Initially, the project was developed using Svelte, but after careful consideration, it was transitioned to jQuery to simplify and optimize the solution.

Backend Development: Python, along with Flask, was employed to create a robust backend. Custom modules were developed to retrieve data from various APIs, including services like Ecowitt and Bluriiot, ensuring seamless data integration.

RTSP Streaming: To overcome the browser limitations regarding RTSP streams, ffmpeg was leveraged to convert the stream into a JPEG sequence. This inventive approach enabled the smooth display of the RTSP stream on the vertical TV.

Configurability: A highly configurable system was designed, allowing for easy adjustments through a configuration file. This file accommodated data types, units, RTSP URL, and credentials for user customization.

User-Friendly Scripts: To simplify installation and uninstallation processes, bash scripts were created. These scripts also added menu entries for user convenience, making it effortless to run and configure the system.

The project resulted in the successful creation of a responsive information display system that met the client's expectations. It effectively integrated data from diverse sources and overcame the challenges associated with RTSP streaming.

The addition of a dark mode option enhanced user experience, and the system's configurability ensures flexibility for future adaptations.

Working on this project enhanced my skills in backend development, data integration, and creative problem-solving. It underscored the importance of adaptability and the ability to choose the right technologies for the task at hand.
