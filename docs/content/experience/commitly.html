Commitly is a cross-platform mobile application developed as part of the CS310 Mobile Application Development course. The project explores how gamification and social accountability can be combined to help users build sustainable habits through shared responsibility and visible progress.

Unlike traditional habit trackers that focus solely on individual progress, Commitly introduces team-based streaks, where members share collective responsibility. This transforms habit formation into a collaborative process rather than a solitary one.

The Challenge

The core challenge was designing a system that balances personal motivation with social dynamics while remaining technically robust across platforms. Implementing reliable streak logic, real-time synchronization, and intuitive UI flows required careful architectural decisions.

Additional complexity came from integrating offline storage with cloud-based synchronization, ensuring consistency between local and remote data sources.

The Solution

Cross-Platform Architecture: The application was developed using Flutter and Dart, enabling a unified codebase for Android and iOS with consistent UI behavior.

Backend Integration: Firebase Authentication and Firestore were used for real-time data synchronization, user management, and team-based interactions. Local persistence was implemented using SQLite for partial offline support.

Gamification System: An XP and streak mechanism was designed to reinforce positive behavior, providing continuous feedback and long-term motivation.

Team Streaks: A shared streak model was implemented where group performance depends on all members, encouraging accountability and peer support.

Quality Assurance: The project included unit and widget tests covering core business logic and critical UI flows, ensuring data integrity and interaction reliability.

Results

The final application delivered a fully functional habit-tracking platform with real-time synchronization, leaderboard support, detailed statistics, and theme customization. Team streaks proved effective in creating sustained engagement among users.

The project demonstrated the feasibility of building scalable, socially driven productivity tools using modern mobile frameworks and cloud services.

My primary responsibility in the team was leading testing and quality assurance, validating business logic, UI behavior, and data consistency across different usage scenarios. This role strengthened my experience in systematic validation and production-level debugging.

Commitly reflects an end-to-end mobile development process, covering system design, cloud integration, automated testing, and collaborative software engineering practices. The source code is available on GitHub.
