Image Metadata Translator

05/05/2021

A client needed a script that could process a folder of images, read their metadata (title, description, keywords), translate the metadata using the DeepL API, and write the translated data back to the images.

Requirements

The project had specific requirements:

  • Allow the user to specify a folder of images, source language, and target language
  • Translate EXIF/IPTC metadata fields: Headline, Description, Keywords
  • Write the new metadata back to the files without modifying filenames
  • Integrate with the DeepL API for translation

Solution

I wrote a Python script that integrated with the DeepL API to automate this manual process. The code was designed to be straightforward and well-documented, making it easy for the client to run on their Mac via Terminal.

The script handled all the specified metadata fields and included proper error handling for edge cases that emerged during development.
